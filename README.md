# MyVision: Customizability meets abstraction

<p align="center">
  <img src="https://github.com/Abhiswain97/MyVision/blob/master/logo.png" height="300" width="300">
</p>

My goal is to make my DL work-flow simple, be ultimately customizable and have also have the right amout of abstraction. So I made **MyVision**. It's my computer vison toolkit. Any functionality I like from the other libraries like PyTorch, fast-ai, keras etc, I integrate it into **MyVision** in a flavour of my own. I will keep updating it, as I learn more things.ðŸ˜„

**MyVision** is in it's nascent stages but improving. You're welcome to contribute ðŸ˜ƒ

## Download:

```
pip install MyVision
```

or if you want to be updated with all the continuous developmental changes just do:

```
git clone https://github.com/Abhiswain97/MyVision.git
cd MyVision
pip install -e .
```
